# 🤖 BerkovStudio Bot v4.0 — Умная система заказов

Профессиональный бот для автоматизации приёма заказов с подтверждением в Telegram.

## 🚀 Возможности v4.0

### ✨ Что нового
- 📬 **Подтверждение клиенту** — автоматическая отправка в Telegram
- 🎫 **ID заказа** — уникальный номер для отслеживания
- 📊 **Трекинг статусов** — new, progress, completed
- ✅ **Чекбокс уведомлений** — клиент выбирает получать ли уведомления
- 🎯 **Выбор типа проекта** — игра, сайт, лендинг, доработка
- 💾 **Сохранение заказов** — все заказы в localStorage
- ⚡ **Индикатор загрузки** — анимация при отправке

## 📋 Настройка

### 1. Создание бота

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot`
3. Имя: `BerkovStudio Bot`
4. Username: `berkovstudio_orders_bot`
5. **Скопируйте токен**

### 2. Получение Chat ID

1. Откройте [@userinfobot](https://t.me/userinfobot)
2. Нажмите **Start**
3. **Скопируйте Chat ID**

### 3. Настройка на сайте

Откройте `script.js` (строки ~170-172):

```javascript
const TG_BOT_TOKEN = '8415434289:AAGyWPDpIIGF19DYIr20Jd4XArtuMLcMRKY';
const TG_CHAT_ID = '5017662184';
```

**Замените на свои значения!**

## 🐛 Решение проблем

### ❌ "Клиент не получает подтверждение"

**Причины:**
1. Клиент не начал диалог с ботом
2. Неправильный username
3. Бот заблокирован

**Решение:**
```
1. Клиент должен написать @berkovstudio_orders_bot
2. Проверьте username в форме
3. Убедитесь что клиент не блокировал бота
```

### ❌ "Заказ не отправляется"

**Проверьте:**
```javascript
// В console.log сайта:
console.log('Token:', TG_BOT_TOKEN);
console.log('Chat ID:', TG_CHAT_ID);

// Должно быть без ошибок
```

### ❌ "Дублирование заказов"

**Причина:** Несколько обработчиков  
**Решение:** Проверьте `script.js` — должен быть **один** `orderForm.addEventListener`
