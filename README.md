# 🎮 BerkovStudio

Современный сайт для приёма заказов на разработку игр и сайтов.

## 🌐 Особенности

- **Современный дизайн** — неоновые цвета, анимации, градиенты
- **Адаптивность** — работает на всех устройствах
- **Форма заказа** — отправка заявок прямо в Telegram
- **Тарифы** — выбор подходящего тарифа для проекта ($1-$100)
- **Анимации** — плавное появление элементов при скролле

## 🛠️ Технологии

- HTML5
- CSS3 (с градиентами и анимациями)
- JavaScript (ванильный)
- Telegram Bot API

## 📦 Как это работает

1. **Выбор тарифа** — клиент выбирает подходящий тариф (Базовый, Стандарт, Премиум)
2. **Заполнение формы** — клиент указывает контакт (Telegram/Discord), желаемую дату готовности и описание проекта
3. **Отправка заявки** — данные автоматически отправляются в Telegram
4. **Связь с клиентом** — я связываюсь с клиентом в указанном мессенджере

## 🚀 Запуск

Просто откройте `index.html` в браузере.

## ⚙️ Настройка Telegram

Для работы формы заказа необходимо настроить Telegram бота:

1. Откройте `script.js`
2. Найдите строки:
   ```javascript
   const TG_BOT_TOKEN = '8415434289:AAGyWPDpIIGF19DYIr20Jd4XArtuMLcMRKY';
   const TG_CHAT_ID = '5017662184';
   ```
3. Замените на свои данные
