# 🎮 BerkovStudio

Современный сайт для приёма заказов на разработку игр и сайтов с мини-играми и таблицей лидеров.

## 🌐 Особенности

### Дизайн и UX
- **Современный дизайн** — неоновые цвета, анимации, градиенты
- **Адаптивность** — полностью работает на всех устройствах (мобильное меню-бургер)
- **Loader** — анимация загрузки при старте страницы
- **Кнопка "Наверх"** — быстрый возврат к началу страницы
- **Плавные анимации** — появление элементов при скролле

### Функционал
- **Форма заказа** — отправка заявок прямо в Telegram
- **Улучшенная валидация** — проверка всех полей перед отправкой
- **Сохранение данных** — автосохранение формы в localStorage
- **Тарифы** — выбор подходящего тарифа ($1-$20)
- **Портфолио** — примеры выполненных работ
- **Отзывы клиентов** — блок с рекомендациями

### Мини-игры
- **🐍 Змейка** — классическая игра с таблицей лидеров
- **💣 Сапёр** — логическая игра с флагами
- **🎰 Слоты** — азартные автоматы с джекпотом
- **🎡 Колесо Фортуны** — крути колесо и выигрывай призы

### Игровая система
- **Таблица лидеров** — топ-10 лучших игроков для каждой игры
- **Баланс и инвентарь** — сохранение прогресса в localStorage
- **Звуковые эффекты** — приятные звуки при действиях
- **Тарифы в инвентаре** — выигрывай и применяй к заказу

## 🛠️ Технологии

- HTML5
- CSS3 (с градиентами, анимациями, backdrop-filter)
- JavaScript (ванильный, ES6+)
- Telegram Bot API
- LocalStorage API
- Canvas API (для игр)
- Web Audio API (звуковые эффекты)

## ⚙️ Настройка Telegram

Для работы формы заказа необходимо настроить Telegram бота:

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Получите токен бота
3. Узнайте свой Chat ID через [@userinfobot](https://t.me/userinfobot)
4. Откройте `script.js`
5. Замените значения:
   ```javascript
   const TG_BOT_TOKEN = 'ВАШ_ТОКЕН_БОТА';
   const TG_CHAT_ID = 'ВАШ_CHAT_ID';
   ```

## 🎮 Как играть в мини-игры

### Змейка
- Управление: стрелки или кнопки на экране
- Цель: собирать еду, расти, не врезаться в стены
- Рекорд сохраняется автоматически

### Сапёр
- Клик левой кнопкой — открыть клетку
- Клик правой кнопкой — поставить флаг
- Цель: найти все мины без взрыва

### Слоты
- Выберите ставку (10-100$)
- Нажмите "Крутить"
- 3 одинаковых символа = джекпот!
- 2 одинаковых = небольшой выигрыш

### Колесо Фортуны
- Крутите колесо за 50$
- Выигрыши от 0$ до 1000$
- Шанс выиграть тариф

## 🏆 Таблица лидеров

Рекорды сохраняются в localStorage браузера:
- 🐍 Змейка — по очкам (10 за каждую еду)
- 💣 Сапёр — по скорости очистки поля
- 🎰 Слоты — по размеру джекпота
